About the Project Sobre o Projeto

ESP32 Connect is a native iOS application built in Swift to provide BLE and Wi-Fi communication with ESP32 boards. The project was created to support electronics education, rapid prototyping and practical automation use cases on Apple devices. ESP32 Connect é um aplicativo iOS nativo desenvolvido em Swift para oferecer comunicação BLE e Wi-Fi com placas ESP32. O projeto foi criado para apoiar ensino de eletrônica, prototipagem rápida e casos práticos de automação em dispositivos Apple.

Features Funcionalidades

  • Native iOS interface for ESP32 communication using Bluetooth Low Energy and Wi-Fi. Interface iOS nativa para comunicação com ESP32 usando Bluetooth Low Energy e Wi-Fi.
  • Reusable control cards for common electronics and robotics peripherals, including digital, analog and PWM interactions. Cartões de controle reutilizáveis para periféricos comuns de eletrônica e robótica, incluindo interações digitais, analógicas e PWM.
  • Support content inside the app, including firmware guidance, serial monitor and electronics learning material. Conteúdo de suporte dentro do app, incluindo orientação de firmware, monitor serial e material de apoio para eletrônica.
  • Designed as a practical platform for education, prototyping and home or lab automation experiments. Projetado como uma plataforma prática para ensino, prototipagem e experimentos de automação em casa ou laboratório.

About the Project Sobre o Projeto

Vindex is a safety-oriented system for motorcyclists that combines fall detection, emergency communication and evidence capture. The architecture integrates embedded electronics, mobile interface and computer vision resources to support a fast response workflow after an accident. Vindex é um sistema voltado à segurança de motociclistas que combina detecção de queda, comunicação de emergência e captura de evidências. A arquitetura integra eletrônica embarcada, interface móvel e recursos de visão computacional para apoiar um fluxo rápido de resposta após um acidente.

Features Funcionalidades

  • Continuous monitoring of movement and accident-related events. Monitoramento contínuo de movimento e eventos relacionados a acidente.
  • Automatic emergency messaging workflow for selected contacts. Fluxo automático de mensagens de emergência para contatos selecionados.
  • Evidence capture pipeline with image processing and plate-recognition support. Pipeline de captura de evidências com processamento de imagem e suporte a reconhecimento de placas.
