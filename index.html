About the Project Sobre o Projeto

Online tools for creating mobile applications, such as MIT App Inventor—widely used in technical schools — do not (as of this project's development in Oct/24) support Bluetooth on Apple devices like the iPhone. To address this issue, this project focuses on creating a native iOS application using Swift that controls all GPIOs of one or multiple ESP32s simultaneously via Bluetooth Low Energy and WiFi. Ferramentas online para criação de apps móveis, como o MIT App Inventor — amplamente utilizado em escolas técnicas — não suportam (até a data de desenvolvimento deste projeto em Out/24) Bluetooth em dispositivos Apple, como o iPhone. Para resolver este problema, este projeto foca na criação de um aplicativo iOS nativo usando Swift para controlar todas as GPIOs de um ou múltiplos ESP32s simultaneamente via Bluetooth Low Energy e WiFi.

Features Funcionalidades

  • The application requires iOS 17 (or higher). O aplicativo requer iOS 17 (ou superior).
  • The control cards are pre-defined for the most common peripherals in educational electronics and robotics. Therefore, this application can be used for various purposes, such as controlling motors, general home appliances, teaching robotics, home automation, etc. Os cartões de controle são pré-definidos para os periféricos mais comuns em eletrônica educacional e robótica. Portanto, este aplicativo pode ser usado para diversos fins, como controlar motores, eletrodomésticos gerais, ensino de robótica, automação residencial, etc.
  • Inside the app, there is a section with the complete firmware to be uploaded to the ESP32 using the Arduino IDE, the pinout for the 30-pin ESP32, a serial monitor, and tutorials for basic electronics projects. Dentro do app, há uma seção com o firmware completo para ser carregado no ESP32 usando a IDE Arduino, o pinout para o ESP32 de 30 pinos, um monitor serial e tutoriais para projetos básicos de eletrônica.
  • The app is rich in control features: analog, digital, and PWM. It was designed to support robotics education and independent home automation. As of Sep/2025, no application with as many features is available for download on any platform. O app é rico em recursos de controle: analógico, digital e PWM. Foi projetado para apoiar o ensino de robótica e automação residencial independente. Até Set/2025, nenhum aplicativo com tantos recursos está disponível para download em qualquer plataforma.

About the Project Sobre o Projeto

Safety device for motorcyclists: in the event of a fall of the motorcyclist, Vindex detects the fall, sends an emergency message to favorite contacts called "Guardians" and with a camera attached takes a photo for visual proof of the accident, being able to recognize vehicle plates for proof of culpability. Dispositivo de segurança para motociclistas: em caso de queda do motociclista, o Vindex detecta a queda, envia uma mensagem de emergência para contatos favoritos chamados "Guardiões" e com uma câmera acoplada tira uma foto para prova visual do acidente, sendo capaz de reconhecer placas de veículos para prova de culpabilidade.

Features Funcionalidades

  • Continuous Monitoring Monitoramento Contínuo
  • Proof of culpability in the event of a collision Prova de culpabilidade em caso de colisão
