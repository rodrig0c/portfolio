iOS App ESP32 Connect
ESP32 Bluetooth iOS/Swift

About the Project

Online tools for creating mobile applications, such as MIT App Inventor—widely used in technical schools — do not (as of this project's development in Oct/24) support Bluetooth on Apple devices like the iPhone. To address this issue, this project focuses on creating a native iOS application using Swift that controls all GPIOs of one or multiple ESP32s simultaneously via Bluetooth Low Energy and WiFi.

Features

  • The application requires iOS 17 (or higher).
  • The control cards are pre-defined for the most common peripherals in educational electronics and robotics. Therefore, this application can be used for various purposes, such as controlling motors, general home appliances, teaching robotics, home automation, etc.
  • Inside the app, there is a section with the complete firmware to be uploaded to the ESP32 using the Arduino IDE, the pinout for the 30-pin ESP32, a serial monitor, and tutorials for basic electronics projects.
  • The app is rich in control features: analog, digital, and PWM. It was designed to support robotics education and independent home automation. As of Sep/2025, no application with as many features is available for download on any platform.
